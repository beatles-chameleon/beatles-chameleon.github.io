# CML 新官网

本仓库为 CML 新官网的源码与文档，项目使用 [VuePress](https://vuepress.vuejs.org/zh/) 搭建。

## 环境

- Node >= 8.6（VuePress 需要）
- [Yarn](https://classic.yarnpkg.com/zh-Hans/)（推荐）

## 起步

`git clone` 本仓库到本地，然后：

```shell script
cd cml-website-new
yarn
```

## 开发

```shell script
yarn dev
open http://localhost:8000
```
