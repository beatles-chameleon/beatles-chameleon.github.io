# cml 新官网

> Next Generation of cml Website.

---

本网站使用 [VuePress](https://vuepress.vuejs.org/zh/) 生成。

## 起步

```shell script
git clone git@git.xiaojukeji.com:cml/cml-website-new.git
cd cml-website-new
```

## 开发

推荐使用 [yarn](https://classic.yarnpkg.com/zh-Hans/)。

```shell script
yarn
yarn dev
open http://localhost:8000
```
