## 导出(Export)

即 将 Chameleon 可重用组件**导出**给普通项目使用，或者说 某端普通项目里面使用 Chameleon 可重用组件。

## 导入(Import)

即 在 Chameleon 可重用项目中**导入**普通项目产出的组件，或者说 可重用代码 Chameleon 项目使用某端普通组件。
